Features

How it Works

1. Ingest

SEC PDFs are parsed and chunked with finance-aware rules.

2. Index

Embeddings stored in FAISS with metadata for source recall.

3. Retrieve

Hybrid search + re-ranking finds the most relevant spans.

4. Answer

LLM synthesizes a concise answer with citations and links.
